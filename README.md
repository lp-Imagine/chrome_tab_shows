# tab-shows

管理浏览器tab选项卡的一个Chrome插件。

# Form

React + Antd + Webpack

## 使用

1. 下载项目

2. 打开Chrome浏览器，地址栏中输入chrome://extensions/，选择加载已解压的扩展程序，并选择下载好的`chrome_tab_shows`目录即可。

3. 快捷键

可以通过`快捷键`方式来启动插件：

|平台|快捷键|
|---|------|
|Windows|`Ctrl`+`Shift`+`Q`|
|MacOS|`Command`+`Shift`+`Q`|
